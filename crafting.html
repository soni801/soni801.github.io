Crafting

All our custom crafting recipes are from Vanilla Tweaks. These are:

Craftable Name Tags

Makes the player able to craft name tags

Craftable Notch Apples

Makes the player able to craft Notch Apples

More Stairs

Creates 8 stairs instead of 4.

Rotten Flesh to Leather

Allows you to smelt rotten flesh into leather

Back to Blocks

Allows you to craft slabs and stairs back into blocks.

Uncraftable Ice

Allows you to break down ice.

Uncraftable Nether Wart Block

Allows you to uncraft nether wart blocks.

Universal Dyeing

Allows you to dye any dyeable block, no matter what colour it is.

Wool to String

Allows you to craft wool into string.
